# Торговец видеокассетами - VHS Tape Trader

Браузерная игра, где вы управляете магазином видеокассет.

## Как играть

1. Откройте `index.html` в браузере (или запустите локальный сервер)
2. Нажмите "Начать день" для открытия магазина
3. Покупатели будут приходить с запросами на фильмы
4. Кликайте на кассеты на стеллаже, чтобы увидеть описание фильма
5. Предлагайте фильм покупателю — если он подходит, вы получите деньги
6. Вечером можно докупить новые фильмы для стеллажа

**Цель:** Продать 50 разных фильмов или 100 фильмов всего.

## Как добавить обложки фильмов

По умолчанию игра использует автоматически сгенерированные обложки-заглушки. Вы можете заменить их на настоящие обложки фильмов.

### Способ 1: Локальные изображения

1. Создайте папку `covers` в корне проекта
2. Добавьте изображения обложек в формате JPG или PNG
3. Рекомендуемый размер: **150x200 пикселей** (соотношение 3:4)
4. Откройте файл `data/films.js`
5. Найдите нужный фильм и измените поле `coverUrl`:

```javascript
// Было (автогенерация):
film.coverUrl = `https://placehold.co/150x200/...`;

// Стало (локальный файл):
{ 
  id: 8, 
  titleRu: "Акира",
  coverUrl: "covers/akira.jpg",  // ← добавьте это поле
  ...
}
```

### Способ 2: Внешние URL

Используйте прямые ссылки на изображения:

```javascript
{ 
  id: 8, 
  titleRu: "Акира",
  coverUrl: "https://example.com/akira-cover.jpg",
  ...
}
```

### Способ 3: Массовое добавление обложек

Отредактируйте функцию генерации в конце файла `data/films.js`:

```javascript
// Замените автогенерацию на свою логику
FILMS.forEach(film => {
    // Например, использовать ID фильма как имя файла:
    film.coverUrl = `covers/${film.id}.jpg`;
    
    // Или транслитерацию названия:
    // film.coverUrl = `covers/${transliterate(film.titleOriginal)}.jpg`;
});
```

### Рекомендации по обложкам

- **Формат:** JPG или PNG
- **Размер:** 150×200px (минимум) — 300×400px (оптимально)
- **Соотношение сторон:** 3:4 (как у VHS кассеты)
- **Имена файлов:** используйте латиницу без пробелов

### Где найти обложки

- [IMDB](https://www.imdb.com/) — поиск по названию фильма
- [MyAnimeList](https://myanimelist.net/) — для аниме
- [The Movie Database (TMDB)](https://www.themoviedb.org/) — API с обложками

## Запуск

Откройте `index.html` напрямую или используйте локальный сервер:

```bash
npx http-server -p 8080
```

Затем откройте http://localhost:8080
